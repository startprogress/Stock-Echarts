## Stock-Echarts

### Description
This project use echarts to show the line chart of a stock prcie in Chinese Stock Market

### Usage
* 1. git clone https://github.com/startprogress/Stock-Echarts.git
* 2. cd Stock-Echarts
* 3. npm install
* 4. supervisor index.js

### Data API
* http://q.stock.sohu.com/
